## 项目实践

### 1. 多模态PDF转Markdown解析模型（强化落地）
- 使用 `pypdf2 + Tesseract OCR` 提取PDF文本与版面信息，结合 **LayoutLMv3** 进行版面解析（标题、段落、表格、图像区域），构建训练数据集。
- 基于 **Qwen-VL** 多模态大模型进行LoRA微调，将OCR文本与版面坐标编码对齐，生成结构化Markdown输出，准确率提升22%。
- 部署为FastAPI服务，支持批量PDF处理，单页推理耗时<500ms。

### 2. 医疗影像分类与数据合成（公开数据集）
- 使用 **ChestX-ray14** 数据集，基于 **ViT-B/16** 预训练模型微调，实现14种胸部疾病的多标签分类，AUC达到0.88。
- 应用 **Stable Diffusion + ControlNet** 生成多样化的病变图像（如肺炎、气胸），扩增稀有类别样本，分类F1提升15%。
- 设计数据合成pipeline：前置病灶分割、后处理质量过滤，确保合成图像医学可用性。

### 3. 多智能体RAG问答系统（复杂编排）
- 基于 **LangChain + AutoGen** 构建Multi-Agent协作系统：检索Agent（FAISS+BM25）、推理Agent（COT）、验证Agent（自我纠错）。
- 集成工具调用链：PDF解析、表格查询、代码执行。在金融财报数据集上实现QA准确率92%，优于单Agent的78%。
- 设计后训练数据生成流程：自动合成“多轮对话+工具调用”样本，用于微调私有Agent模型。

### 4. 小说续写模型（COT推理强化）
- 使用 **Qwen-2-7B**，在《白鹿原》语料上进行LoRA+DeepSpeed ZeRO-3微调，支持指定风格续写。
- 引入 **COT思维链** 数据：生成剧情推理链（如“人物关系→动机→情节发展”），微调后模型叙事逻辑性显著提升，人工评分提高37%。
- 训练pipeline集成RLHF（DPO）优化偏好对齐，消除重复和逻辑跳跃。

### 5. AI英语对话系统（Agent场景）
- 设计咖啡厅、机场等8种教学场景，基于 **DeepSeek** 定制化提示词，实现上下文记忆与角色扮演。
- 构建工具调用链：查询词汇、纠错语法、生成例句。系统支持主动提问引导，提升学习交互性。

### 6. AI简历生成系统（数据处理+算法设计）
