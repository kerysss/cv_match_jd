## 项目实践

### 1. 多模态语义搜索与向量检索系统（2026.03 - 2026.04）
- **目标**：实现基于CLIP/Qwen的图文语义搜索，支持大规模数据的难例挖掘与相似度检索。
- **技术栈**：CLIP + Qwen-2-7B + Faiss + Milvus + FastAPI + Docker
- **核心工作**：
  - 使用CLIP提取图像/文本embedding，融合Qwen的语义表征，设计特征融合策略。
  - 在Milvus中构建向量索引，支持Top-K相似度搜索与余弦距离度量。
  - 实现基于Recall/Precision的检索精度评估，优化embedding质量。
  - 部署RAG管线，将检索结果输入LLM进行生成式问答，探索生成式搜索在数据挖掘中的应用。
- **成果**：在10万级图文数据集上达到85%+的Top-5检索准确率，支持实时查询响应。

### 2. AI简历生成系统（2026.01 - 2026.03）
- **目标**：自动抓取电商平台招聘职位并生成定制化简历，底层逻辑可类比数据挖掘中的匹配与排序。
- **技术栈**：Scrapy + Qwen-2-7B + LangChain + PyTorch + Spark
- **核心工作**：
  - 抓取1000+大模型相关职位，设计职位-简历的多维匹配打分系统（文本相似度、技能覆盖度等）。
  - 基于Qwen模型生成定制简历内容，使用LangChain实现多轮上下文管理与结构化输出。
  - 使用Spark对职位数据进行ETL与特征工程，支撑实时匹配推理。
- **成果**：系统上线后生成简历匹配率提升40%，证明在数据挖掘与匹配排序场景的工程能力。

### 3. AI英语对话系统（2026.02 - 2026.03）
- **目标**：构建基于RAG的英语教学对话系统，支持场景化交互与上下文记忆。
- **技术栈**：DeepSeek + LangChain + ChromaDB + FastAPI
- **核心工作**：
  - 构建知识库（咖啡厅、机场等场景对话模板），使用embedding存储至ChromaDB向量库。
  - 设计RAG检索增强生成流程，根据用户问题检索最相关场景模板，结合上下文生成回答。
  - 实现对话历史记忆管理，支持多轮交互。
- **成果**：系统在50个用户测试中对话自然度评分4.5/5，验证了RAG在数据挖掘与智能检索中的可行性。

### 4. 小说续写模型（2026.01 - 2026.02）
- **目标**：基于白鹿原小说数据微调Qwen-2-7B，实现风格一致的文本续写。

- **成果**：生成文本风格匹配度达90%以上，展示了端到端的大模型微调与部署能力。
