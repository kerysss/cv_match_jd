  - 使用Flink构建实时特征流，实现秒级延迟的点击归因与模型特征更新。
  - 将数据写入ClickHouse与Iceberg，支持分析师与模型训练的高效查询。
- **成果**：模型训练数据覆盖率提升30%，实时特征延迟降低至5秒以内。

- **职责**：
  - 设计数仓分层架构（ODS/DWD/DWS/ADS），完成数据模型设计。
  - 使用Iceberg实现数据湖存储，支持ACID事务与增量读取。
  - 开发ETL调度任务（Spark SQL + Airflow），确保数据准确性与时效性。
- **成果**：数仓覆盖全链路广告效果指标，数据一致性达99.9%。

### 3. 流量广告效果归因实时数据开发（腾讯广告部门）
- **目标**：实现广告点击、转化归因的实时统计与业务展示。
- **职责**：

- **成果**：归因系统延迟低于3秒，支撑每日亿级事件处理。

### 4. AI简历生成系统（个人项目）
