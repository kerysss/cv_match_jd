## 项目实践

### 1. AI简历生成系统（大模型应用 + RAG + 定制化生成）
- 从Boss直聘抓取1000+大模型招聘职位，设计**职位-简历匹配打分系统**，利用大模型意图理解提取职位核心要求。
- 基于匹配结果，调用LLM生成定制化简历内容，实现**自动化多步骤推理与工具调用**（ReAct框架：分析职位 → 检索知识库 → 生成内容 → 校验）。
- **技术亮点**：RAG知识库构建（职位描述向量化+语义检索）、Prompt优化、多轮对话生成。
- **与职位关联**：直接对应“用户意图理解、通⽤问答、智能对话导购”场景，展示了大模型在招聘场景中的决策与生成能力。

### 2. 小说续写模型（SFT + LoRA + DeepSpeed微调）
- 使用《白鹿原》小说数据，基于Qwen-2-7B开源模型，通过**LoRA + DeepSpeed ZeRO-3**进行指令微调，训练出符合原作文风的长文本续写模型。
- 设计数据清洗、格式模板、训练策略，在单机多卡环境下完成全流程训练。
- **与职位关联**：展示SFT、LoRA、DeepSpeed的实际落地能力，与职位“模型微调、Prompt优化”要求高度匹配。

### 3. AI英语对话系统（RAG + Agent + 上下文记忆）
- 构建咖啡厅、旅行、商务等英语教学场景，定制化Prompt模板，实现**多轮上下文记忆**（基于ConversationBufferWindowMemory）。
- 集成DeepSeek API作为基座模型，利用**Function Calling**调用外部词典/语法检查工具，实现智能纠错与场景引导。
- **与职位关联**：体现RAG、Agent、Function Calling在对话系统中的应用，可直接复用到“智能对话导购、商家互动智能助⼿”场景。

### 4. 狗狗皮肤病检测模型（传统机器学习 + 图像分类）
- 收集6种狗狗皮肤病的图片数据集，使用XGBoost（图像特征提取+分类）训练皮肤病类型识别模型，验证精度达92%。
- **与职位关联**：展示多模态数据处理能力及传统ML经验的快速迁移能力，可为“多模态商品检索、AIGC素材生成”提供图像理解基础。
