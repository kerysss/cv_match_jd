## 项目实践

### 1. 多模态职位匹配与简历生成系统（Agent + RAG + 多模态检索）
- **目标**：针对大模型算法工程师岗位，自动抓取BOSS直聘1000+职位，构建多模态匹配引擎，生成定制化简历。
- **技术栈**：CLIP（文本-图像跨模态编码）、Sentence-BERT（语义检索）、LLM（Qwen-2）生成、RAG（检索增强生成）、Agent（自动决策流程）
- **核心贡献**：设计多模态打分系统，融合职位描述文本与公司Logo/办公环境图像，通过CLIP计算跨模态相似度，结合LLM生成匹配理由；利用RAG从简历库中召回最相关经历，实现智能简历定制化。

### 2. 小说续写模型（大模型微调 + RLHF对齐）
- **目标**：基于《白鹿原》语料，使用Qwen-2-7B模型，训练小说风格续写模型。
- **技术栈**：LoRA微调、DeepSpeed ZeRO-3、SFT、PPO（RLHF）
- **核心贡献**：完成从数据清洗（长文本分块）、指令模板设计、LoRA微调（单卡A100 80GB）、到RLHF对齐（使用偏好数据对进行PPO训练）的全流程；最终模型在连贯性、风格一致性上超越原始模型，理解长文本上下文依赖。

### 3. AI英语对话系统（提示词工程 + 对话记忆 + Agent）
- **目标**：构建咖啡厅等真实场景的多轮英语对话教学系统。
- **技术栈**：DeepSeek API、LangChain、上下文记忆（向量数据库）、提示词工程（Chain-of-Thought、角色扮演）
- **核心贡献**：设计结构化提示词模板，实现语音输入→LLM生成→语法纠错→发音反馈的闭环；利用记忆机制保持对话上下文一致性，用户满意度提升30%。

### 4. 狗狗皮肤病检测模型（图像分类 + 迁移学习）
- **目标**：基于6类皮肤病图片数据集，训练多分类模型。
- **技术栈**：XGBoost（特征工程）、CNN（ResNet50迁移学习）、数据增强（旋转/翻转/色彩抖动）
- **核心贡献**：对比传统机器学习（XGBoost + HOG/SIFT手工特征）与深度学习（ResNet50）方案，最终使用预训练CNN + 微调达到92%准确率；项目锻炼了图像数据处理、模型评估与调优能力。
