## 项目实践

### 1. AI英文对话系统（直接匹配「对话系统」要求）
- **目标**：为英语学习者提供沉浸式对话练习，支持咖啡厅、机场、面试等多场景
- **技术栈**：DeepSeek API + 定制化提示词工程 + 上下文记忆（滑动窗口+摘要压缩）
- **亮点**：
  - 设计多角色对话提示模板，实现场景化、个性化的自然对话生成
  - 引入错误纠正与语法反馈机制，提升教学实用性
  - 封装为REST API，使用FastAPI + 异步处理，支持并发调用
- **成果**：完成20+场景对话流，用户测试满意度92%

### 2. 小说续写模型（直接匹配「自然语言生成」要求）
- **目标**：基于《白鹿原》小说数据，训练能够续写风格一致的文本生成模型
- **技术栈**：Qwen2-7B + LoRA + DeepSpeed ZeRO-3 + 文本数据清洗（NLTK分词、去噪）
- **亮点**：
  - 构造高质量续写训练对（前文-后文），采用data augmentation增强多样性
  - 使用spaCy进行实体识别与风格特征提取，辅助模型捕捉叙事结构
  - 在单机8卡A100上训练，loss降至0.8，续写文本风格相似度达87%
- **成果**：产出可交互的续写demo，可生成长达2000字的连贯段落

### 3. AI简历生成系统（体现NLP应用与职位匹配能力）
- **目标**：抓取Boss直聘上1000+大模型相关职位，自动匹配并生成定制化简历
- **技术栈**：Scrapy + Qwen-72B API + 向量数据库（Faiss） + 提示词模板
- **亮点**：
  - 利用RAG技术检索JD关键要求，结合候选人背景生成针对性简历叙述
  - 集成职位匹配打分系统（基于语义相似度与技能关键词匹配）
- **成果**：简历匹配度平均提升40%，缩短求职准备时间80%

### 4. 狗狗皮肤病检测模型（体现传统ML能力，补充NLP之外的算法广度）
- **目标**：识别6种常见犬类皮肤病
- **技术栈**：XGBoost + 图像特征提取（ResNet预训练）+ 数据增强
- **成果**：准确率84%，部署为轻量级API
