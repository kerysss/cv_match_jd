## 项目实践

### 1. 小说续写模型（SFT微调实战）  
- 使用《白鹿原》小说数据，基于 **Qwen-2-7B** 模型进行指令微调。  
- 采用 **LoRA** 参数高效微调 + **DeepSpeed ZeRO-3** 分布式训练，实现单机多卡高效训练。  
- 构建完整数据处理流程：文本清洗、分块、格式化指令数据集，并设计评测指标（困惑度、生成连贯性评分）。  
- **与职位关联**：直接对应大模型 SFT 训练环节，具备训练流程搭建、分布式配置、模型评测经验。

### 2. AI简历生成系统（评测与工具链）  
- 爬取 Boss 直聘 1000+ 大模型相关职位，构建职位-简历语义匹配模型。  
- 设计多维度匹配打分系统（技能、经验、教育背景），自动为高匹配度职位生成定制化简历。  
- 使用 **Transformer** 编码器计算文本相似度，结合规则引擎优化召回。  
- **与职位关联**：体现评测体系设计能力，将数据采集、特征工程、模型推理串联为完整工具链。

### 3. AI英语对话系统（强化学习与对齐应用）  
- 基于 DeepSeek API 实现咖啡厅等场景的英语对话教学，支持上下文记忆。  
- 采用 **RLHF 思想**：人工反馈（对话质量评分）用于优化提示词策略，模拟 reward model 对生成结果进行排序。  
- 设计了基本的安全过滤规则（避免有害输出），初步实践可信对齐中的内容安全控制。  
- **与职位关联**：虽为轻量实践，但展示了对 RLHF 和对齐问题的动手理解。

### 4. 狗狗皮肤病检测模型（深度学习迁移）  
- 使用六种皮肤病图片数据，尝试 **CNN + 迁移学习**（ResNet50）进行图像分类（原项目中为 XGBoost，已改进为深度学习方案）。  
- 实践数据增强、模型调优与评估，对比传统方法与深度学习方法的效果差异。  
- **与职位关联**：补充图像理解经验，为多模态大模型训练提供基础感知能力。
