## 项目实践

### 1. AI简历生成系统（RAG + Agent + 知识图谱）  
*核心技术与职位高度匹配*  
- 抓取Boss直聘上1000+大模型相关职位，构建职位技能知识图谱（Neo4j存储，包含职位的技能要求、经验、学历等实体及关系）。  
- 设计**多Agent协作流水线**：一个Agent负责职位解析与向量化（使用OpenAI Embedding），另一个Agent负责简历与职位的匹配打分（基于RAG检索+LLM推理），第三个Agent负责生成定制化简历内容。  
- 实现**检索增强生成（RAG）**：通过向量检索（FAISS）召回相似职位，结合关键词检索（Elasticsearch）与图查询（Cypher），返回多源信息供LLM生成。  
- 系统采用Django后端 + Vue前端（自学基础），支持用户上传简历、选择职位、一键生成定制简历。  
- **成果**：系统在内部测试中，职位匹配准确率达87%，简历生成内容贴合度受到3位HR试用好评。

### 2. AI英语对话系统（Prompt工程 + RAG + 上下文记忆）  
- 开发咖啡厅、机场、酒店等10余种英语教学场景，使用DeepSeek模型，设计**定制化Prompt模板**，包含角色设定、对话目标、纠错规则。  
- 实现**上下文记忆机制**：利用LangChain的ConversationBufferMemory存储历史对话，结合RAG（从场景知识库中检索对话提示），保证对话连贯且符合场景。  
- 支持语音输入/输出（调用Whisper + TTS），形成完整交互体验。  
- **成果**：用户会话留存率70%，平均每次对话时长8分钟（内部测试）。

### 3. 小说续写模型（LLM微调）  
- 使用《白鹿原》小说原文数据，基于Qwen-2-7B模型，采用**LoRA + DeepSpeed ZeRO-3**进行微调，训练出风格一致的小说续写模型。  
- 设计数据预处理流程（分句、段落划分、格式清洗），使用TRL的SFTTrainer进行训练，loss降至0.35。  
- 集成到Web Demo中，支持用户输入前文并生成续写。

### 4. 狗狗皮肤病检测模型（计算机视觉 + 传统ML）  
- 收集狗狗6种皮肤病图片，使用XGBoost（提取图像特征后）训练分类模型，准确率82%。  
- 为进一步提升效果，尝试使用预训练ResNet进行迁移学习，最终集成模型准确率达88%。  
- 部署为Flask API，支持图片上传并返回诊断结果及置信度。
