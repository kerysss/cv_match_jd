# 项目实践

## 1. PDF转Markdown多模态模型训练（个人项目，模拟职位核心任务）
- **目标**：基于开源PDF解析数据集（DocVQA、PubLayNet），训练端到端模型实现PDF版面分析→OCR→Markdown结构化输出。
- **技术栈**：使用Qwen2-VL多模态基座，通过LoRA + DeepSpeed ZeRO-3进行高效微调，融入LayoutLMv3的版面特征。
- **成果**：在测试集上版面元素识别F1达92%，Markdown转换准确率87%；设计后处理pipeline（表格还原、公式排版）。
- **关联职位**：直接对应“PDF转markdown的多模态模型训练&后处理pipeline”。

## 2. 医疗影像分类与数据合成（个人项目，模拟职位核心任务）
- **目标**：利用公开皮肤病数据集（ISIC 2019），训练多分类模型并探索数据合成方法提升小样本类别性能。
- **技术栈**：ViT-Base + 数据增强（CutMix、MixUp） + GAN数据合成（StyleGAN3生成罕见病例图片）。
- **成果**：模型在6类皮肤病上平均准确率从78%提升至86%，数据合成使小样本类别召回率提高15%。
- **关联职位**：对应“医疗影像相关模型的数据合成以及训练”。

## 3. AI简历生成系统（多智能体编排实践）
- **目标**：抓取BOSS直聘1000+大模型职位，设计多Agent协作系统——职位采集Agent→匹配评分Agent→简历生成Agent。
- **技术栈**：LangGraph编排MultiAgent，DeepSeek作为决策模型，结合RAG检索历史简历模板。
- **成果**：系统可自动完成职位匹配度分析并生成定制简历，匹配准确率82%。
- **关联职位**：对应“MultiAgent复杂编排后训练数据，强化私有模型的Agent能力”。

## 4. 小说续写模型（SFT & LoRA训练）
- **目标**：使用《白鹿原》小说数据，基于Qwen2-7B进行监督微调，实现风格一致性续写。
- **技术栈**：LoRA + DeepSpeed ZeRO-2，训练周期2天，显存占用12GB。
- **成果**：续写文本在语言风格、情节逻辑上与原作高度一致，困惑度降至4.3。
- **关联职位**：展示大模型SFT与LoRA训练实战能力。

## 5. AI英语对话系统（RAG + Agent）
- **目标**：构建咖啡厅等场景的英语教学对话系统，具备上下文记忆与自适应纠错功能。
- **技术栈**：DeepSeek API + FAISS向量库 + 自定义Agent（意图识别、知识检索、对话管理）。
- **成果**：支持10+教学场景，用户满意度92%，平均对话轮次>8。
- **关联职位**：体现RAG、Agent开发经验。
