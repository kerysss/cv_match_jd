## 项目实践

### 1. AI简历生成系统（RAG + Agent全链路设计）
- **技术栈**：LangChain, Qwen-2-7B, PyTorch, DeepSpeed, vLLM, 百度爬虫
- **项目描述**：爬取Boss直聘10万+职位数据，构建职位匹配打分系统（BERT语义匹配+规则融合），设计Agent智能体：用户输入个人信息 → Agent调用职位匹配工具 → 自动选取高匹配职位 → 调用LLM生成定制简历文档。
- **亮点**：
  - 实现了多工具调用链（职位检索→打分→简历生成）的Agent全链路。
  - 使用vLLM高吞吐推理引擎，单次生成延迟<2s。
  - 引入上下文记忆管理，支持多轮对话修改简历。
- **关联职位需求**：直接对应Agent全链路搭建，具备工具调用、记忆管理、RAG检索、LLM微调实践。

### 2. 小说续写模型（LLM后训练与微调）
- **技术栈**：Qwen-2-7B, LoRA, DeepSpeed ZeRO-2, FlashAttention, TRL
- **项目描述**：使用《白鹿原》语料，对Qwen-2-7B进行LoRA微调（SFT），训练出风格一致的续写模型。训练中采用DeepSpeed ZeRO-2进行分布式优化，FlashAttention加速长序列训练，loss收敛至0.12。
- **亮点**：
  - 掌握从数据准备→tokenize→训练→推理的全流程。
  - 实践了LoRA低秩适配和DeepSpeed混合精度训练。
  - 对比了SFT与RLHF（DPO）的效果差异。
- **关联职位需求**：对应“模型研发与训练优化”，显示对LLM微调、分布式训练、优化技术的实战能力。

### 3. AI英语对话系统（Agent对话全链路搭建）
- **技术栈**：DeepSeek-7B, LangChain, 上下文记忆模块, 场景Prompt设计, 语音交互（TTS/ASR）
- **项目描述**：针对咖啡厅、酒店等场景，设计Agent对话智能体：用户语音输入 → ASR → LangChain Agent（多轮上下文记忆+场景Prompt）→ LLM生成回答 → TTS输出。实现了上下文剪枝、场景切换、对话历史管理。
- **亮点**：
  - 独立设计Agent对话全链路，包含状态追踪、工具调用（场景切换/知识检索）、记忆压缩。
  - 通过定制化Prompt提升了对话连贯性与场景适应性。
- **关联职位需求**：对应“产品智能体全链路搭建”，展示了从输入到输出的完整Agent链路设计能力。

### 4. 狗狗皮肤病检测模型（视觉分类基础）
- **技术栈**：XGBoost, CNN特征提取, PIL图像处理
- **项目描述**：收集6种狗狗皮肤病图片，提取HOG/颜色直方图特征，使用XGBoost分类模型，准确率88%。后尝试使用预训练ResNet进行微调，准确率提升至92%。
