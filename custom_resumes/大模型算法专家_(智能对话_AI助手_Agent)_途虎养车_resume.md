## 项目实践

### 1. 汽车养护智能导购Agent（自研项目，模拟途虎业务场景）
**核心目标**：构建面向汽车养护场景的对话Agent，实现商品推荐、故障诊断引导及多轮状态跟踪。  
**技术选型**：Qwen-2-7B 基座 + LoRA微调 + LangChain Agent + ChromaDB RAG  
**关键实现**：
- **全链路数据**：爬取途虎养车公开产品库、维修案例、常见问答，构建40万条高质量结构化数据集，完成去重、清洗、标注（意图+槽位+对话状态）。
- **Agent架构**：设计三层路由（意图识别→知识检索→工具调用），集成商品数据库（SQL）、故障诊断API（模拟）、库存查询等工具；使用ReAct模式实现工具链自动编排。
- **多轮对话状态跟踪**：基于槽位填充+历史压缩，解决上下文遗忘与指代消解；在500轮压测中任务完成率从62%提升至89%。
- **效果验证**：建立A/B Test框架，对比基座模型与微调模型在意图识别准确率（+18%）、推荐点击率（+12%）和用户满意度（+0.7分）上的表现。
- **部署**：vLLM + AWQ 4bit量化实现单卡100+并发推理，平均延迟＜300ms。

### 2. AI英语对话教学系统
**目标**：开发沉浸式英语教学对话机器人，支持多场景（咖啡厅、机场等）自由对话。  
**技术**：DeepSeek API + 自定义提示词模板 + 上下文记忆管理（滑动窗口+摘要压缩）  
**亮点**：设计动态Prompt（含角色、语法纠错、难度自适应）；实现对话历史非结构化存储与检索，支持最长50轮上下文一致性。

### 3. 小说续写模型（SFT + DeepSpeed）
**目标**：基于《白鹿原》数据集训练LLM续写能力。  
**流程**：文本预处理（分词、段落分割）→ Qwen-2-7B + LoRA微调 → DeepSpeed ZeRO-3分布式训练（8卡A100）→ 生成评估（perplexity、BLEU）。  
**成果**：训练损失收敛至1.2，生成文本在风格一致性上表现良好。

### 4. AI简历生成系统
**目标**：自动抓取BOSS直聘大模型岗位，生成定制简历。  
**技术**：Scrapy抓取1000+职位 → 基于BERT的匹配打分模型 → 模板化简历生成（LLM + Prompt）。  
**价值**：理解招聘JD与候选人能力映射，为简历优化提供数据驱动策略。
