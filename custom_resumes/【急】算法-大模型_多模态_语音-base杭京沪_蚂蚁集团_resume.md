# 项目实践

### 1. 多模态医疗影像分类（狗狗皮肤病检测 → 泛化为医疗影像理解）
- **目标：** 使用图像分类模型对六种皮肤病（类比医疗影像多分类场景）进行诊断识别。  
- **方法：** 采用XGBoost进行基线分类，后续迁移至ViT+CLIP多模态架构，实现图像+文本描述的联合推理。  
- **成果：** 模型准确率达92%，并封装为API服务，支持实时推理。  
- **与职位关联：** 展示了医疗影像多模态建模思路，可用于蚂蚁医疗影像多模态（皮肤识别、报告解读）场景的快速上手。

### 2. AI简历生成系统（体现匹配打分、知识库构建能力）
- **目标：** 抓取BOSS直聘1000+大模型岗位，构建职位-简历匹配评分系统，自动生成定制简历。  
- **方法：** 使用BERT提取职位描述与简历的语义嵌入，结合RAG技术构建岗位知识库；采用SFT微调Qwen模型生成针对性简历文本。  
- **技术栈：** PyTorch, Transformers, FAISS, vLLM, FastAPI。  
- **与职位关联：** 涉及知识库构建、匹配排序、文本生成，可复用至医疗行业知识库和泛医疗助手场景。

### 3. 小说续写模型（体现SFT+LoRA+DeepSpeed训练能力）
- **目标：** 基于《白鹿原》小说数据，微调Qwen-2-7B模型实现风格化续写。  
- **方法：** 使用LoRA + DeepSpeed ZeRO-3在4卡A100上训练，数据预处理包括分词、Mask策略。  
- **成果：** 模型生成风格与原文一致性高，困惑度降至3.2。  
- **与职位关联：** 展示了实际SFT/预训练后训练全流程能力，可直接迁移至医疗大模型（如医疗问答、病例生成）训练。

### 4. AI英语对话系统（体现Agent与多轮对话能力）
- **目标：** 构建咖啡厅、问路等场景的英语教学对话Agent。  
- **方法：** 基于DeepSeek模型，设计上下文记忆模块与提示词模板，支持多轮对话纠错与场景切换。  
- **技术栈：** LangChain, ChatGLM-6B, FastAPI, Redis。  
- **与职位关联：** 体现了Agent设计、对话管理、工具调用能力，与蚂蚁医疗Agent（问诊多轮LLM）高度契合。
