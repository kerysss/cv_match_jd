## 项目实践

### 1. AI英语对话教学Agent（可迁移至客服/运营场景）

- **目标**：构建多场景英语教学对话Agent，替代人工教师完成口语陪练。
- **技术栈**：LangChain + DeepSeek API + 上下文记忆模块 + 自定义工具调用
- **关键实现**：
  - 设计咖啡厅、旅行、商务等6种教学场景，通过定制化Prompt驱动Agent行为逻辑。
  - 实现对话历史检索与长期记忆管理（RAG + 向量数据库），使Agent能根据用户历史错误针对性教学。
  - 引入工具调用链：当用户询问实时信息（如天气、菜单）时，Agent自动调用外部API获取数据并返回。
- **成果**：Agent交互流畅度达90%+，可独立完成20分钟以上教学对话，用户留存率提升30%。
- **与职位的关联**：此项目完整复刻了客服Agent的核心能力——场景化Prompt、工具集成（API/数据库/RAG）、上下文理解与自主决策，可直接迁移至塔斯汀的订单查询、库存咨询、工单分类等业务场景。

### 2. Agent驱动的AI简历生成系统

- **目标**：从招聘平台抓取职位信息，自动分析匹配度并生成定制化简历。
- **技术栈**：Scrapy + LLM（Qwen-2-7B） + RAG + 匹配评分Agent
- **关键实现**：
  - 抓取BOSS直聘1000+大模型相关职位，构建职位画像库。
  - 设计多维度匹配评分Agent：基于LLM对候选人技能、经验、项目与职位描述进行语义匹配，输出结构化评估报告。
  - 调用LLM根据匹配报告生成定制化简历，自动调整措辞与重点。
- **成果**：匹配准确率85%+，简历生成时间从人工2小时缩短至3分钟。
- **与职位的关联**：体现了将业务需求（职位匹配）转化为Agent训练目标与评估指标的能力，与塔斯汀“工单分类、信息提取、自动回复”的业务转化思路一致。

### 3. 小说续写模型（LLM微调全流程）

- **目标**：基于白鹿原语料，训练具备风格延续能力的文本生成模型。
- **技术栈**：Qwen-2-7B + LoRA + DeepSpeed + Transformers
- **关键实现**：
  - 数据清洗与预处理：对白鹿原全集进行分句、标点校正、段落结构化。
  - 使用LoRA在4块A100上进行参数高效微调，结合DeepSpeed ZeRO-3优化训练效率。
  - 设计续写评估指标（风格一致性、情节连贯性），通过少量样本学习迭代优化。
- **成果**：续写文本风格相似度达92%，生成速度<2s/段落（vLLM部署）。
- **与职位的关联**：展示了完整的LLM微调、优化与部署流程，与Agent训练中“通过微调提升准确率与用户体验”直接匹配。

### 4. 狗狗皮肤病检测模型（分类任务工程化）

- **目标**：对6种常见狗狗皮肤病进行图像分类。
- **技术栈**：XGBoost + 图像特征工程 + Flask部署
- **关键实现**：提取图像纹理、颜色直方图等特征，训练集成模型，准确率88%；封装为REST API，支持批量推理。
